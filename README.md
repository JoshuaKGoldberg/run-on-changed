<h1 align="center">Run on Changed Files</h1>

<p align="center">
	Runs a command on all files transitively impacted by changes in JS/TS files.
	🎯
</p>

## Usage

```shell
npm i run-on-changed-files
```

```ts
import { greet } from "run-on-changed-files";

greet("Hello, world! 💖");
```
